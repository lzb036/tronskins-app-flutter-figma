# 文档规范

## 核心原则

注释解释「**为什么**」，而非「是什么」。代码本身说明做了什么，注释说明背后的意图、约束和决策原因。

```dart
// ❌ 无意义：重复代码
// 获取用户信息
final user = await getUserInfo();

// ✅ 有价值：说明原因
// 服务器切换后需重新拉取，本地缓存已失效
final user = await getUserInfo();
```

## Dart 文档注释（`///`）

Dart 使用 `///` 作为文档注释（等价于 JSDoc），**不使用** `/** */`。

### 必须添加 `///` 的场景

| 场景 | 说明 |
|------|------|
| 公开类 | 所有 `class`、`mixin`、`enum`、`extension` |
| 公开方法 / 函数 | 包括 `static` 方法 |
| 公开属性 / 字段 | 特别是响应式状态（`.obs`）和计算属性 |
| 构造函数参数 | 含义不明显的 `required` 参数 |
| 复杂逻辑块 | 算法、状态机、特殊边界处理 |

### 格式规范

```dart
/// 市场列表控制器，负责管理皮肤列表的加载、筛选和排序。
///
/// 监听 [GlobalGameController.currentAppId] 变化，切换游戏时自动刷新数据。
/// 分页状态由内部私有字段管理，外部通过 [fetchMarketList] 触发加载。
class MarketController extends GetxController {
  /// 当前皮肤列表，响应式，UI 通过 Obx 订阅
  final RxList<MarketItem> items = <MarketItem>[].obs;

  /// 是否正在加载，用于防止重复请求和显示加载指示器
  final RxBool isLoading = false.obs;

  /// 加载市场列表数据
  ///
  /// [reset] 为 true 时重置分页并清空现有数据，用于下拉刷新场景。
  /// 加载中时调用会直接返回，防止并发请求。
  Future<void> fetchMarketList({bool reset = false}) async { ... }
}
```

### 参数说明

参数说明写在正文中，用 `[paramName]` 引用，**不使用** `@param` 标签：

```dart
/// 格式化价格为当前货币显示字符串。
///
/// [usdAmount] 必须是 USD 计价的原始金额，内部会按当前汇率换算。
/// 负数、NaN、Infinity 均返回 `'$ 0'`。
String format(double usdAmount) { ... }
```

### 返回值说明

有非显而易见返回值时，在正文中说明：

```dart
/// 尝试取消提现申请。
///
/// 返回 `true` 表示取消成功，`false` 表示服务端拒绝（如已处理中）。
Future<bool> cancelWithdraw(String id) async { ... }
```

### 引用其他符号

用 `[SymbolName]` 引用类、方法、属性，IDE 会生成超链接：

```dart
/// 切换游戏后触发 [refreshAll]，确保数据与 [GlobalGameController.appId] 一致。
```

## 行内注释（`//`）

用于解释复杂逻辑、非显而易见的决策、临时处理：

```dart
// 先更新内存态，确保界面能即时响应；再异步持久化，避免 UI 卡顿
currentAppId.value = nextAppId;
await GameStorage.setGameType(nextAppId);

// 服务端可能无限返回重复分页，用 Set 去重后若无新数据则停止加载
final uniqueItems = list.where((item) => _seenKeys.add(item.id)).toList();
if (uniqueItems.isEmpty) {
  _hasMore = false;
  return;
}
```

## TODO / FIXME 标记

```dart
// TODO: 待后端支持批量接口后，改为单次请求
// FIXME: 极端情况下汇率为 0 会导致除零，需服务端保证数据合法性
// HACK: WebView Session 注入依赖私有 API，Flutter 升级后需验证
```

格式固定为 `// TODO:` / `// FIXME:` / `// HACK:`，后跟说明原因的中文描述。

## API 接口函数注释模板

```dart
/// 获取市场物品列表。
///
/// [params] 包含 appId、分页和筛选条件，appId 由调用方从
/// [GlobalGameController] 获取后传入，禁止在此函数内部读取。
///
/// 返回空列表而非 null，调用方无需判空。
Future<List<MarketItem>> getMarketItemList(MarketListParams params) async { ... }
```

## 控制器注释模板

```dart
/// 钱包控制器，管理充值、提现、资金流水和锁定余额。
///
/// 资金相关操作（充值、提现）完成后会调用 [refreshUser] 同步最新余额。
/// 分页列表统一通过 `reset: true` 参数触发刷新，避免重复封装。
class WalletController extends GetxController {
  /// 资金流水列表，按时间倒序
  final RxList<WalletFundFlowItem> fundFlows = <WalletFundFlowItem>[].obs;

  /// 是否还有更多流水记录可加载
  bool get hasMoreFundFlows => _fundFlowHasMore;
}
```

## 模型注释模板

```dart
/// 市场物品实体，对应服务端 `/market/items` 接口返回的单条数据。
@JsonSerializable()
class MarketItem {
  /// 物品唯一 ID，用于详情页跳转和收藏操作
  final String id;

  /// USD 计价的当前售价，展示时通过 [CurrencyController.format] 转换
  final double price;

  /// 磨损度数值，范围 0.0 ~ 1.0，展示时使用 [ItemWearLabel] 组件
  final double? wear;
}
```

## Widget 注释模板

```dart
/// 市场物品卡片，展示物品图片、名称、价格和磨损度。
///
/// 价格展示依赖外部传入的已格式化字符串，组件内部不做货币换算。
/// 点击事件通过 [onTap] 回调交由父级处理，组件不直接导航。
class MarketItemCard extends StatelessWidget {
  /// 物品数据
  final MarketItem item;

  /// 已格式化的价格字符串，由 [CurrencyController.format] 生成
  final String formattedPrice;

  /// 点击卡片时的回调，为 null 时卡片不可点击
  final VoidCallback? onTap;
}
```

## 禁止事项

- 禁止无意义注释（注释内容与代码完全重复）
- 禁止使用 `/** */` 多行注释（Dart 惯例是 `///`）
- 禁止英文注释（项目统一使用中文，技术术语可保留英文原文）
- 禁止注释掉大段废弃代码提交到仓库，应直接删除
- 禁止 `// ignore:` 抑制整个文件的 lint 规则，只允许单行抑制并附说明
