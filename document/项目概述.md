# 项目概述

## 项目定位

TronSkins 是一款面向 CS:GO 皮肤交易的 Flutter 移动端市场应用，连接买家与卖家，深度集成 Steam 进行身份验证和库存管理，支持 Android 和 iOS 双平台，并通过 Shorebird 实现 OTA 热更新。

## 技术栈

| 领域 | 技术 | 说明 |
|------|------|------|
| UI 框架 | Flutter（Dart SDK `^3.9.2`） | 强制使用，Material 3（`useMaterial3: true`） |
| 状态管理 & DI | GetX `^4.6.1` | 响应式状态用 `.obs`，控制器继承 `GetxController`，路由级控制器只通过 Binding 注册 |
| 本地持久化 | GetStorage `^2.0.5` | 轻量 KV 存储，必须通过 `lib/common/storage/` 类型化封装访问 |
| 网络 | Dio `^5.3.2` | 单例 `HttpHelper`，拦截器链顺序固定：Header → Auth → Cookie → Logging |
| 序列化 | json_annotation `^4.9.0` + json_serializable `^6.8.0` | 生成 `.g.dart`，修改模型后必须重新运行 build_runner |
| 不可变模型 | freezed_annotation `^2.4.4` + freezed `^2.4.4` | 生成 `.freezed.dart`，生成文件提交到仓库 |
| 值相等 | equatable `^2.0.5` | 用于模型值比较 |
| 图片缓存 | cached_network_image `^3.3.0` | 远程图片统一使用，禁止裸 Image.network |
| 列表交互 | flutter_slidable `^3.1.0` | 可滑动列表项 |
| HTML 渲染 | flutter_html `^3.0.0` | 渲染富文本/公告内容 |
| 图表 | fl_chart `^0.65.0` | 价格趋势图 |
| 二维码 | qr_flutter `^4.1.0` / mobile_scanner `^4.0.0` | 展示与扫描 |
| WebView | webview_flutter `^4.10.0` | Steam 登录，涉及 Session 注入，修改需谨慎 |
| 图片上传 | image_picker `^1.0.4` | 用户头像等图片选取 |
| 安全 & 加密 | encrypt `^5.0.1` + dart_sm `^0.1.5` | SM2/SM4 加密，封装在 `lib/common/security/` |
| 生物识别 | local_auth `^2.1.6` | 指纹/面容认证 |
| 2FA | otp `^3.0.0` | TOTP 验证码生成 |
| 国际化 | intl `^0.18.0` | 日期/数字格式化，17 种语言翻译在 `lib/l10n/` |
| OTA 热更新 | shorebird_code_push `^2.0.5` | 通过 Shorebird 发布补丁，无需重新上架 |
| 日志 | logger `^1.4.0` | 结构化日志，描述信息使用中文 |
| 静态分析 | flutter_lints `^5.0.0` | `flutter analyze` 检查，单行抑制用 `// ignore: rule_name` |

## 核心功能模块

| 模块 | 路径 | 说明 |
|------|------|------|
| Marketplace（市场） | `lib/pages/market/` | 浏览、购买、出售皮肤；筛选、排序、价格趋势图 |
| Inventory（库存） | `lib/pages/navbar/inventory.dart` | 查看 Steam 库存，将物品上架到商店 |
| Shop（商店） | `lib/pages/shop/` | 卖家管理挂单、定价、订单、批量购买请求 |
| Wallet（钱包） | `lib/pages/wallet/` | 充值、提现、结算、资金流水、锁定余额 |
| Steam 集成 | `lib/pages/steam/` | WebView 登录、交易链接绑定、Session 注入、2FA/TOTP |
| User（用户） | `lib/pages/user/` | 个人资料、收藏、消息通知、设置 |
| Help Center（帮助中心） | `lib/pages/help/` | FAQ 分类、反馈工单创建与跟踪 |
| Loyalty（积分/优惠） | `lib/pages/integral/`, `lib/pages/coupon/` | 积分系统、优惠券管理、抽奖 |

## 关键全局控制器

| 控制器 | 位置 | 职责 |
|--------|------|------|
| `GlobalGameController` | `lib/common/hooks/game/` | 管理当前选中游戏（appId），所有 API 请求必须携带 |
| `CurrencyController` | `lib/common/hooks/currency/` | 统一货币格式化，页面层禁止直接处理金额 |
| `UseTheme` | `lib/common/hooks/theme/` | 主题切换 |
| `UseLocale` | `lib/common/hooks/locale/` | 语言切换 |

以上控制器均在 `main()` 中以 `Get.put(..., permanent: true)` 注册。

## 常用命令

```bash
flutter pub get                                              # 安装依赖
flutter analyze                                             # 静态分析
flutter test                                                # 运行所有测试
dart run build_runner build --delete-conflicting-outputs    # 重新生成模型代码
flutter build apk                                           # Android release APK
flutter build ios                                           # iOS release 构建
shorebird release android                                   # 发布 OTA 补丁（Android）
shorebird release ios                                       # 发布 OTA 补丁（iOS）
```
